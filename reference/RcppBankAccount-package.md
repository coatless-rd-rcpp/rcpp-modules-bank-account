# RcppBankAccount: Expose the BankAccount C++ Class using Rcpp Modules

Provides an implementation of a BankAccount 'C++' class being connected
to 'R' using 'Rcpp Modules' to expose the underlying class methods.

## Details

RcppBankAccount is a package showing how to use Rcpp Modules to make
available a C++ class under S4.
